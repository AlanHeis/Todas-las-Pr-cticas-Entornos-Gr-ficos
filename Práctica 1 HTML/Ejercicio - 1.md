# 1. 
HTML es un lenguaje de marcas de hipertexto. Es un lenguaje de marcado para la elaboración de páginas web. Fue creado por Tim Berners-Lee en 1991. Pasó por las versiones 1.0, 2.0, 3.2, 4.01, XHTML y la actual, HTML5.

# 2. 
El W3C recomienda:
- Separar el contenido (HTML) de la presentacion (CSS).
- Garantizar la accesibilidad para que todos los suarios puedan navegar.
- Asegurar la interoperabilidad para que funcione en cualquier navegador y/o dispositivo.
- Uso correcto de etqieutas semánticas.
- Código valido y bien estructurado.

# 3.
Desaprobado: un elemento que todavía funciona pero no se recomienda usar.

Obsoleto: un elemento que ya no se debe usar y no se asegura que los navegadores lo sigan mostrando bien.

# 4.
El DTD define las reglas que debe seguir un documento HTML, sirve tambien para comunicarle al navegador qué versión de HTML se está usando. 

En HTML 4.01 existen tres tipos:
- Strict: solo estructura, sin elementos de presentación.
- Transitional: permite etiquetas antiguas.
- Frameset: para páginas con frames.

# 5.
Son etiquetas que dan información sobre la página que no se ven directamente en la pantalla. Se definen con la etiqueta `<meta>` dentro del `<head>`
